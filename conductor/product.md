# Product Definition - BookingCore-ES

## Project Description
Motor de reservas escalable diseñado específicamente para prevenir el overbooking bajo carga extrema.

## Problem Statement
Latencia inaceptable y bloqueos de base de datos que interrumpen el flujo de ventas.

## Target Users
Equipos técnicos de cadenas hoteleras gestionando picos de demanda estacionales.

## Key Goals
- Garantizar 0% de overbooking en entornos de alta concurrencia mediante Redlock.
- Mantener tiempos de respuesta de reserva < 200ms bajo carga sostenida.
